EDENA Bootstrap Premium Website Template

Made with by Little Neko

Based on Twitter Bootstrap 3, a sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development.

Along with all the Bootstraps components, we added some extra shortcodes to help you customize your website, easily insert carousel, or call to actions boxes and many more.

100 + templates

EDENA is build with easy customization in mind. Out of the box, you get more than 30 page templates, 10 color schemes and multiple home page layouts variants, from corporate to agency portfolio, including a one page version...

All layouts are made of sections, that you can easily combine to fit your specific project. Just copy and paste the block you need to easily build your layout!

Mobile first, responsive and retina ready

All graphic elements are build using CSS3, for a sharp look on any device.

Icon font take a set of icons or pictograms that would normally be implemented as an image or vector file and then convert it into a font for a faster loading time than images, scalability and endless variations using CSS.
